# fastapi-foundation
Learning FastAPI fundamentals including routing, Pydantic validation, CRUD operations, and dependency injection.

## Features Implemented

- FastAPI setup with uv
- SQLAlchemy integration (SQLite)
- Pydantic schemas for validation
- Dependency Injection for DB sessions
- Full CRUD operations:
  - Create User
  - Get All Users
  - Get User by ID
  - Update User (PATCH)
  - Delete User
